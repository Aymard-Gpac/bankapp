(inner) normalizeFrequency(frequency) → {"once"|"weekly"|"monthly"}

Normalise la fréquence envoyée par le frontend.

Règle :

  • "once" = exécution immédiate
  • "weekly" = transaction récurrente hebdomadaire
  • "monthly" = transaction récurrente mensuelle
  • toute autre valeur => "once"
